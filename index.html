PumpChumps

Pump Chumps is a game my friends and I developed in the summer of 2021. It is a third-person "Gas Station Simulator" with 3d graphics and online coop. I worked mainly as a programmer along with contributing to the shaders and map design. We built the game using Unity, and the Photon Pun package for the networking. I learned a lot from this project and got a lot of positive feedback from players.
